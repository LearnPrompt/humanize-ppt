# Agent 从入门到精通 Hermes 版

这是一次 Humanize PPT 展示样例，包含中文 guizang 稳定路径和英文 Neo-Grid 多风格选择路径。

## 在线路径

合并到 `main` 并由 GitHub Pages 发布后：

- 中文 PPT：`https://learnprompt.github.io/humanize-ppt/showcase/hermes-agent-mastery/ppt/`
- 中文演讲模式：`https://learnprompt.github.io/humanize-ppt/showcase/hermes-agent-mastery/presenter/`
- 英文 Neo-Grid PPT：`https://learnprompt.github.io/humanize-ppt/showcase/hermes-agent-mastery/en/ppt/`
- 英文 Neo-Grid 演讲模式：`https://learnprompt.github.io/humanize-ppt/showcase/hermes-agent-mastery/en/presenter/`

## 本地路径

- `ppt/index.html`：完整 guizang Swiss HTML PPT。
- `presenter/index.html`：讲者控制台。
- `presenter/notes.json`：逐页讲者备注。
- `presenter/presenter_manifest.json`：演讲模式产物记录。
- `en/ppt/index.html`：英文 Neo-Grid Bold 完整 deck。
- `en/presenter/index.html`：英文讲者控制台。
- `en/materials/`：GPT Image 2 静态素材、Remotion 视频素材和素材生产记录。
- `en/qa/qa_report.md`：英文路径 QA 记录。

## 说明

这个样例固定两条推荐路径：中文内容优先走 guizang 稳定成稿，再补素材 QA、演讲模式和部署；英文内容先做 5 个风格候选，选中风格后再进入完整 deck、演讲模式和部署。
